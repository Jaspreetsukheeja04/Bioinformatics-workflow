# Bioinformatics Workflow

This repository contains practical bioinformatics workflows focused on molecular docking, transcriptomics (RNA-seq), and protein–protein interaction (PPI) network analysis.

## Workflows

- Molecular Docking: Protein–ligand interaction using CB-Dock2 and AutoDock Tools  
- Transcriptomics: RNA-seq data analysis using Galaxy, FastQC, Trim Galore, and HISAT2  
- Cytoscape: PPI network construction and hub gene identification using STRING and Cytoscape  

## Tools Used

UCSF Chimera, AutoDock Tools, CB-Dock2, Galaxy, FastQC, MultiQC, Trim Galore, HISAT2, STRING, Cytoscape
